# Documentation Prompts

## Function Documentation
```
Generate comprehensive documentation for this function:
{paste function}

Include:
- Purpose description
- Parameters with types and descriptions
- Return value with type and description
- Exceptions that can be thrown
- Usage examples
- Notes about edge cases or performance
Use {JSDoc/Javadoc/Python docstring/etc.} format.
```

## Class Documentation
```
Document this class thoroughly:
{paste class}

Include:
- Class purpose and responsibility
- Constructor parameters
- Public methods with full documentation
- Properties/attributes
- Usage examples
- Design patterns used (if any)
```

## API Documentation
```
Create API documentation for this endpoint:
{paste endpoint code}

Include:
- Endpoint description
- HTTP method and path
- Request parameters (query, path, body)
- Request body schema
- Response format and status codes
- Error responses
- Authentication requirements
- Example requests and responses
```

## README Generation

```
Create a README.md for this project:

- Contributing guidelines

```
