# Learning and Explanation Prompts

## Concept Explanation
```
Explain {concept} in {language/technology}.

Include:
- What it is
- Why it's used
- How it works
- Code examples
- Common use cases
- Best practices
- Common pitfalls
```

## Code Explanation
```
Explain how this code works:
{paste code}

Break down:
- Overall purpose
- Each major section
- Key algorithms or patterns used
- Time and space complexity
- Edge cases handled
```

## Technology Comparison
```
Compare {technology A} vs {technology B} for {use case}.

Compare:
- Strengths and weaknesses
- Performance characteristics
- Learning curve
- Community and ecosystem
- Use cases best suited for each
- Recommendation for my scenario
```

## Best Practices
```
What are the best practices for {topic} in {language/framework}?

Include:
- Industry standards
- Common patterns
- Things to avoid
- Code examples
- Security considerations
- Performance implications
```

## Step-by-Step Tutorial
```
Create a step-by-step tutorial for {task/feature}.

Include:
- Prerequisites
- Setup instructions
- Detailed steps with code
- Explanations for each step
- Common errors and solutions
- Next steps for learning
```

## Error Understanding
```
I don't understand this error message:
{paste error}

In the context of:
{paste code or describe context}

Please explain:
- What the error means
- What causes it
- How to fix it
- How to prevent it in the future
```

## Design Pattern Learning
```
Teach me about the {design pattern name} pattern.

Include:
- Problem it solves
- Structure and components
- Real-world example
- Code implementation in {language}
- When to use it
- When not to use it
```

## Code Review Learning
```
Review this code as if teaching a junior developer:
{paste code}

Provide:
- What's done well
- What could be improved
- Why each suggestion matters
- Alternative approaches
- Learning resources
```
