# Code Generation Prompts

## Basic Function Generation
```
Write a {language} function that {description of functionality}.
Requirements:
- Input: {input parameters}
- Output: {expected output}
- Include error handling
- Add inline comments
```

## Class Generation
```
Create a {language} class called {ClassName} that {purpose}.
Include:
- Constructor with parameters: {parameters}
- Methods: {list of methods}
- Properties: {list of properties}
- Documentation comments
```

## API Endpoint
```
Generate a REST API endpoint in {framework} that:
- Method: {GET/POST/PUT/DELETE}
- Route: {/path}
- Accepts: {input format}
- Returns: {output format}
- Include validation and error handling
```

## Data Structure
```
Implement a {data structure name} in {language} with the following operations:
- {operation 1}
- {operation 2}
- {operation 3}
Include time and space complexity analysis in comments.
```

## Algorithm Implementation
```
Implement the {algorithm name} algorithm in {language}.
Input: {input description}
Output: {output description}
Optimize for: {time/space/readability}
Include test cases.
```

## Database Query
```
Write a SQL query to {description of what to retrieve/update}.
Tables involved: {table names}
Join conditions: {if applicable}
Filters: {conditions}
Return: {columns to return}
```

## Web Component
```
Create a {React/Vue/Angular} component that {functionality}.
Props/Inputs: {list}
State: {list}
Events: {list}
Include TypeScript types and styling.
```

## Utility Function
```
Write a utility function in {language} that {purpose}.
Make it:
- Pure function (no side effects)
- Well-tested
- Handle edge cases: {list edge cases}
- Return type: {type}
```
